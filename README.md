## Лабораторна робота №3: Об'єктно-орієнтоване програмування в Java
Реалізація консольного додатка на Java з використанням принципів ООП (успадкування, поліморфізм, інкапсуляція) та архітектурного шаблону **MVC (Model-View-Controller)** для обробки геометричних фігур.

#### Завдання (Варіант 3.1)
Програма має моделювати набір геометричних фігур (`Rectangle`, `Triangle`, `Circle`) та виконувати наступні операції:
- Розрахунок площі кожної фігури.
- Підрахунок сумарної площі всіх фігур.
- Сортування фігур за **площею** (зростання).
- Сортування фігур за **кольором** (алфавітний порядок).

#### Model (Модель даних)
- **`Drawable` (Interface):** Визначає поведінку малювання методом `draw()`.
- **`Shape` (Abstract Class):** Базовий клас для всіх фігур. Реалізує `Drawable`, зберігає колір та визначає абстрактний метод `calcArea()`.
- **`Circle`, `Rectangle`, `Triangle`:** Конкретні класи, що реалізують математику розрахунку площі.

#### View (Відображення)
- **`ShapeView`:** Відповідає за виведення текстової інформації та масивів фігур у консоль користувача.

#### Controller (Керування)
- **`ShapeController`:** Містить бізнес-логіку програми, масив даних фігур та методи для маніпуляції ними (сортування, підрахунок суми).

#### Використані технології та інтерфейси
- **`Comparator`:** Використовується для гнучкого сортування об'єктів за різними критеріями без зміни коду самих класів фігур.
- **`Java Arrays.sort`:** Для ефективного впорядкування масивів.
- **`String.format`:** Для красивого виведення результатів розрахунків площ.

## Лабораторна робота №8: Багатопотокове програмування
Проект реалізує паралельну обчислювальну систему для знаходження наближеного значення числа **π**. Використовується статистичний метод Монте-Карло, де випадкові точки генеруються в одиничному квадраті, а результат базується на відношенні точок, що потрапили в межі вписаного кола.

##### Параметри обчислень
*   **Загальна кількість ітерацій:** 1,000,000,000 (1 мільярд).
*   **Вхідний аргумент:** Кількість потоків (Threads).
*   **Вивід:** Значення PI, кількість потоків, загальна ітерація та час виконання (ms).

##### Як запустити
1. Скомпілюйте проект:
   javac src/*.java
2. Запустіть програму, вказавши кількість потоків (наприклад, 8):
   java -cp src ParallelMonteCarloPi 8
